## HU01 – Encendido y apagado de dispositivos

Como usuario del hogar,
quiero poder encender o apagar luces, cámaras y electrodomésticos desde una consola o panel,
para tener control sobre el consumo y el confort.

## HU02 – Modos de funcionamiento automáticos

Como propietario del hogar,
quiero activar modos predefinidos (Ecológico, Vacaciones, Seguridad Nocturna),
para que el sistema ajuste automáticamente luces, climatización y seguridad según el contexto.

## HU03 – Sensores inteligentes

Como sistema del hogar,
quiero recibir notificaciones de los sensores de movimiento, temperatura o humedad,
para ejecutar acciones automáticas como encender luces o activar alarmas.

## HU04 – Control energético

Como usuario,
quiero visualizar el consumo energético de cada dispositivo,
para optimizar el gasto y detectar dispositivos ineficientes.

## HU05 – Seguridad doméstica

Como usuario,
quiero recibir alertas cuando se detecta movimiento o una puerta se abre sin autorización,
para mantener la seguridad del hogar.

## HU06 – Control remoto unificado

Como administrador del hogar,
quiero poder monitorear y controlar todos los dispositivos desde un único punto,
para gestionar el hogar fácilmente y evitar inconsistencias.

## HU07 – Registro y administración de usuarios

Como propietario,
quiero registrar diferentes usuarios con roles (admin, familiar, invitado),
para definir permisos sobre qué dispositivos puede controlar cada uno.

## HU08 – Modo de emergencia

Como sistema,
quiero ejecutar un protocolo de emergencia cuando se detecta fuego, humo o intrusión,
para alertar a los ocupantes y activar medidas de seguridad.
